mini-mongodb
===========

       模仿mongodb采用Xml+json实现小型数据库;
        1.实现数据库创建
        2.表的创建
        3.表数据的增、删、改、查


-----------------------------------
       使用方法：
       1、首先导入该XmlDAO.Jar包
       2、导入Jdom包,提供解析xml文件功能
       3、导入Gson包,提供解析json文件功能
       4、用XmlDAO类里的方法生成一个xml的数据库文件
       5、测试表数据的增、删、改、查功能
